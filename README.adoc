== Compiling

=== From a container

1. Create HTML

 sudo docker run -it -v `pwd`:/documents/:Z asciidoctor/docker-asciidoctor make html

1. Clean up files

 sudo docker run --rm -v `pwd`:/workdir:Z fedora/container-best-practices make clean
